This project demonstrates how to automate LinkedIn posts using GitHub Actions and the LinkedIn API.

### Features
- Triggers every day at the set time.
- Extracts up to 4 commits data from the last 3 repos with recent activity.
- Posts sumarized formatted updates to LinkedIn using Hugging Face AI.
- Uses OAuth 2.0 and GitHub Secrets to handle authorizations and API keys.

### Technologies
- Node.js
- GitHub Actions
- LinkedIn REST API
